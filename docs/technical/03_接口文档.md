## 前端

### `init.ready`

前端初始化完成后发送，表示可以正常接收数据了。

无参数。

### `modelID.update`

用户选择的新的模型，向插件端确认。

参数：

- `modelID: string` 新选择的模型 ID

## 插件端

### `models.update`

参数：

- `models: string` 模型配置的字符串
- `modelID: string` 选中的模型的ID

### `modelID.updated`

插件端确认了新选择的模型，通知前端修改。

参数：

- `modelID: string` 新选择的模型 ID
